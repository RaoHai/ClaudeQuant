# Quote

获取实时行情 - 显示指定股票的详细行情数据

## Description

获取指定股票的实时行情信息，包括最新价、涨跌幅、开盘价、最高价、最低价、成交量、成交额等详细数据。

## Arguments

- `<stock_code>` - 股票代码（必需）
  - 格式：不需要带市场后缀（.SH/.SZ）
  - 示例：`600519`（贵州茅台）、`000858`（五粮液）、`601318`（中国平安）

## Dependencies

- Python 3.9+
- pandas
- akshare

## Configuration

无需配置，AkShare 完全免费。只需要在 `.env` 文件中配置持仓代码：
```env
PORTFOLIO_SYMBOLS=600519,000858,601318
```

## Examples

```bash
/quote 600519
/quote 000858
/quote 601318
```

## Natural Language Triggers

Claude 会在以下情况自动调用此 skill：

- "查看600519的行情"
- "贵州茅台现在多少钱"
- "帮我看看贵州茅台的股价"
- "600519最新价格是多少"

## Exit Codes

- `0` - 成功获取行情数据
- `1` - 缺少股票代码参数或数据获取失败

## Error Messages

如果缺少参数，会显示：
```
❌ 错误：缺少股票代码

用法: /quote <股票代码>

示例：
  /quote 600519    # 贵州茅台
  /quote 000858    # 五粮液
  /quote 601318    # 中国平安

💡 提示：代码不需要带市场后缀（.SH/.SZ），系统会自动识别
```

## Notes

- 系统会自动识别股票市场：
  - 6开头 → 上海证券交易所（.SH）
  - 0/3开头 → 深圳证券交易所（.SZ）
  - 4/8开头 → 北京证券交易所（.BJ）
- 使用 AkShare 数据源，来自东方财富网
- 完全免费，无需注册或 Token
