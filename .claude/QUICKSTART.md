# 快速开始 - ClaudeQuant

5分钟快速上手 ClaudeQuant AI 持仓分析助手。

## 第一步：安装依赖

```bash
pip install -r requirements.txt
```

安装的主要包：
- `pandas` - 数据处理
- `tushare` - 行情数据
- `click` - 命令行工具
- `python-dotenv` - 环境变量

## 第二步：配置 Tushare Token

### 获取 Token

1. 访问 https://tushare.pro/
2. 注册账号（免费）
3. 登录后在"个人中心"获取 Token

### 配置环境变量

编辑 `.env` 文件：

```env
# 填入你的 Tushare Token
TUSHARE_TOKEN=你的token在这里

# 配置你的持仓股票（逗号分隔）
PORTFOLIO_SYMBOLS=600519,000858,601318
```

**注意**：
- 股票代码不需要带市场后缀（.SZ/.SH），系统会自动识别
- 深圳股票以 0/3 开头，上海股票以 6 开头，北交所以 4/8 开头

## 第三步：测试 Skills

### 测试持仓概况

```bash
./skills/portfolio.sh
```

应该看到类似输出：
```
📊 持仓概况

代码         名称      最新价    涨跌幅
----------------------------------------
600519.SH   贵州茅台  1650.00   +2.50%
```

### 测试单只股票行情

```bash
./skills/quote.sh 600519
```

### 测试技术分析

```bash

```

### 生成完整报告


报告会保存在 `reports/` 目录。

## 常见问题

### 1. Tushare 提示"没有接口访问权限"

**原因**：部分接口需要积分

**解决**：
- 完成注册任务获取积分
- 或使用免费的实时行情接口（代码已处理）

### 2. 持仓列表为空

**检查**：
- `.env` 文件中 `PORTFOLIO_SYMBOLS` 是否配置
- 股票代码格式是否正确（不要带 .SZ/.SH 后缀）
- 例如：`PORTFOLIO_SYMBOLS=600519,000858` ✅
- 错误：`PORTFOLIO_SYMBOLS=600519.SH,000858.SZ` ❌

### 3. Skills 无法执行

**检查**：
- Skills 脚本是否有执行权限：`chmod +x skills/*.sh`
- Python 环境是否正确：`which python3`

### 4. 报告生成失败

**检查**：
- `reports/` 目录是否存在
- 是否有写入权限
- Tushare 数据是否获取成功
